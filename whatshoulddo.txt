1.git add 文件 添加即将提交的文件
2.git status看状态
3.git diff 文件 看区别
4.git commit -m "这里是本次的描述"
5.git reset --hard HEAD^ 或者指定的标号 退回
6.git reflog  看清楚所有提交的版本，可以回归未来
7.git checkout -- file 用版本库change掉当前的工作区
8.git rm XX
9关联一个远程库，使用命令git remote add origin git@server-name:muxiaokui/repo-name.git
关联后，使用命令git push -u origin master 不允许就-f强制

10. git checkout -b dev 创建一个分支并切换 等于git branch dev && git checkout dev 不知道现在head分支用git branch查
11. 合并分支git merge dev 将dev合并到当前  git branch -d dev 删除
